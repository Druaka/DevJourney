This MEAN stack web app is made possible thanks to free-tier cloud hosting—services that let you run apps and databases online at no cost. It’s a simple way to learn, test, and launch your ideas.

🛠️ Backend

Hosted on Render (free-tier), which automatically redeploys the backend whenever changes are pushed to the devjourney Git repository's backend directory. Render spins down the backend after 15 minutes of inactivity to conserve resources. When a new request comes in, the service spins back up—this can take up to a minute, causing a brief delay before the app responds. Powered by Node.js and Express, the backend exposes RESTful endpoints to serve Pokémon TCG data, fetched and processed using TCGdex (a dedicated Pokémon TCG API). The server begins listening for requests only after the routes and database are successfully initialized.

🗄️ Database

Hosted on OVH MongoDB Discovery. During backend startup, the server connects to the database. Once connected, it drops the existing database, fetches the latest Pokémon card sets via the TCGdex SDK, sanitizes the data, stores it in MongoDB, and caches it in memory for fast access.
