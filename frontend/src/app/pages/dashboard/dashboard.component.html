This MEAN stack web app is made possible thanks to free-tier cloud hosting—services that let you run apps and databases online at no cost. It’s a simple way to learn, test, and launch your ideas.

🛠️ Backend

Hosted on Render’s  free-tier, the backend is automatically redeployed whenever changes are pushed to the devjourney repository’s backend directory. Due to the limitations of the free-tier, the service spins down after 15 minutes of inactivity. When a new request is made, it spins back up—this cold start can take up to a minute, introducing a short delay before the app responds. Built with Node.js and Express , the backend exposes RESTful endpoints to serve Pokémon Trading Card Game (TCG) data, fetched and processed using TCGdex , an API dedicated to Pokémon TCG data. The server begins listening for requests only after the routes and database are successfully initialized.

🗄️ Database

Hosted on OVHcloud MongoDB Discovery . During backend startup, the server connects to the database. Once connected, it fetches the latest Pokémon card sets via the TCGdex SDK, sanitizes the data, stores it in MongoDB , and caches it in memory for fast access.
